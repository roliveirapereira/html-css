Principais formatações

Negrito

Existem 2 maneiras de colocar negrito usando a tag bBsem semantica ou tendo um termo em destaque usando a tag STRONG

Italico

Tambem existem 2 maneiras para colocar um texto em italico usando a tag I sem semantica ou usando a tag EM com semantica

Texto marcado

Podemos criar um texto marcado ultilizando a tag MARK

Texto sublinhado

Podemos usar o INS para sublinhar um texto com semantica ou o U sem semantica

Texto sobrescrito

Podemos usar a tag SUP para sobscrever um texto como x3=20

Texto subescrito

Podemos usar a tag sub para subscrever um texto como H2O
